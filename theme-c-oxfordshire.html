The Quiet English Art of Saying Everything by Saying Nothing

· · ·
There is a particular genius in the English way of not saying things. It is misread, constantly, by everyone who doesn't grow up inside it. The nod that means disapproval. The "quite" that means catastrophic. The invitation for tea that means "I'd like you to leave now, and possibly forever." This is not dishonesty. It is something more interesting — a language of implication so precise that it conveys more than direct speech could. The problem, if it is one, is that you have to be inside the code to read it. Outsiders see reserve. Insiders see an entire emotional library, cross-referenced and indexed.
